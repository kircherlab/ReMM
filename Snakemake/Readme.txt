Important files in the snakemake folder:

- Snakemake file
- file with snakemake rules for frature download: feature_download

- folder Utils with 
	- Template of config file for feature download: features_config.json
	- Template of config file for PARSMURF: scaffold.json
	- Excel list of features: Features.xlxs

- folder Scripts with helper scripts
	- generate_parsmurf_config.py
	- convert_data.R

Folders input and output are created and filled by snakemake
